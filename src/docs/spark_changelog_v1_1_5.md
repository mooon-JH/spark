# v1.1.5 UX 개선 업데이트

## ✅ 수정된 5가지

### 1. 제목/본문 구분선 추가
**Before**: 제목과 본문이 바로 이어져서 구분 어려움
**After**: 제목 아래 얇은 회색 구분선 추가

```
제목
───────────  ← 새로 추가!
본문
```

---

### 2. 스캔 애니메이션 설명
**Before**: 우측 상단에 "글을 읽고 있어요" (위치가 이상함)
**After**: 본문 중앙 상단에 "피드백을 위해 글을 분석하고 있어요" (검은 둥근 배지)

```
┌─────────────────────┐
│                     │
│  피드백을 위해 글을  │  ← 중앙에 명확하게
│  분석하고 있어요     │
│                     │
│   [블러된 본문]      │
│   ~~~~~~~~~~~       │
└─────────────────────┘
```

---

### 3. 제목도 블러 처리
**Before**: 피드백 모드에서 제목은 선명, 본문만 블러 (이상함)
**After**: 제목부터 본문까지 전체 블러 처리

---

### 4. 구조 배너 개선
**Before**: 
```
[회색 박스]
"글 전체의 흐름을..."
```
→ 인용구처럼 보여서 피드백인지 모름

**After**: 
```
[주황색 박스]
💡 글 흐름 파악
"글 전체의 흐름을..."
```
→ 아이콘 + 명확한 타이틀

---

### 5. "다음 시도" 문구 변경
**Before**: 
```
[회색 박스]
다음에 시도해볼 것
"~~를 더 구체적으로..."
```
→ "뭘 시도하라는거지?" 헷갈림

**After**: 
```
[파란색 박스]
✍️ 이렇게 써보면 어떨까요?
"~~를 더 구체적으로..."
```
→ 제안/조언임을 명확히 전달

---

## 🎨 시각적 변화

### 구조 배너 (글 흐름)
- 색상: zinc-50 → amber-50
- 테두리: 검정 → 주황 (amber-500)
- 아이콘: 없음 → 💡
- 타이틀: 없음 → "글 흐름 파악"

### 다음 제안 박스
- 색상: zinc-50 → blue-50
- 테두리: 회색 → 파랑 (blue-500)
- 아이콘: 없음 → ✍️
- 타이틀: "다음에 시도해볼 것" → "이렇게 써보면 어떨까요?"

---

## 📱 사용자 경험 개선

1. **명확한 구조**: 제목/본문이 시각적으로 분리
2. **직관적 스캔**: 중앙 배지로 "지금 분석 중"임을 명확히 인지
3. **일관된 블러**: 전체 페이지가 피드백 모드임을 느낌
4. **Tip 스타일**: 아이콘 + 타이틀로 각 섹션이 무엇인지 즉시 이해
5. **친근한 문구**: "이렇게 써보면 어떨까요?" → 조언처럼 느껴짐
