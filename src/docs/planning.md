# ✦ Spark — MVP 기획안

> 버전 2.3 | 2025년  
> Co-writing 글쓰기 앱 | "생각에 불을 붙이는 글쓰기"

---

## 컨텍스트 인계

너는 전문적인 PM이자 개발자야. MVP를 만들기 위해 최대한 단순하고 간소화된 방향으로 개발해야 해. 내 의견을 항상 비판적으로 분석해야 해.

## 1. 기획 진화 흐름

글감 기획이 어떤 문제의식으로 어떻게 발전했는지 맥락을 정리한 섹션입니다.

| 단계 | 핵심 변화 | 문제 / 이유 |
|---|---|---|
| 1단계 초기 설계 | 카테고리 10개 (스타트업·성장·커리어 등) / 결핍·후회 중심 톤 | 부정적 감정 편향 / 수필 느낌 / 카테고리 중복 |
| 2단계 타겟 재정의 | 직장인 한정 → 예비 창업자·내면을 가꾸고 싶은 사람으로 확장 | 카테고리와 글감의 결 전면 재검토 필요 |
| 3단계 글감 설계 원칙 | 긍정·호기심·탐구 톤 확립 / 서술형+질문형 병행 / 구체적 장면 포착 | 목적 추가: 글쓰기 습관 + 기획력·생각 구조화 훈련 |
| 4단계 카테고리 재편 | 10개 → 16개 (4개 축으로 재구성) | 중복 통합·분리·확장·제외 다수 결정 |
| 5단계 프롬프트 방향 | 단일 프롬프트 → 카테고리별 16개 개별 설계 | 각 카테고리마다 톤·예시·금지어 따로 설계 |

## 2. 제품 개요

### 핵심 가치
- 시작을 쉽게 — 글감 + 첫 문장이 질문형식으로 제시됨
- 이탈하지 않게 — 편리한 UX로 흐름 유지
- 쌓이게 — 글이 누적되어 습관화

### 타겟 사용자
| 구분 | 내용 |
|---|---|
| 연령대 | 20대 중반 ~ 30대 중반 |
| 타입 1 | 스타트업 종사자 / 커리어 성장에 관심 있는 직장인 |
| 타입 2 | 창업을 희망하는 사람 (막연히 꿈꾸는 직장인 + 구체적으로 준비 중인 예비 창업자) |
| 타입 3 | 나의 내면을 잘 가꾸어내고 싶은 사람 |
| 페인포인트 | 내 생각을 구조화 시키는데 어려움이 있어, 글쓰기 연습을 통해 길러내고 싶은 사람 |

### 비전 로드맵
| 단계 | 내용 |
|---|---|
| MVP (지금) | 글감 + 첫 문장으로 쓰기 시작 → 습관 형성 |
| v2 | 3초 멈춤 → 다음 질문 제안 → 생각 전개 훈련 |
| v3 | 글 구조 피드백 (주장·근거·구조화) → 기획력 훈련 |

## 3. 카테고리 (16개 / 4축)

| 축 | 카테고리 | 핵심 질문 |
|---|---|---|
| 비즈니스 (5개) | 커리어 | 일하는 나는 어떤가 |
| | 창업 | 내가 만들고 싶은 것 |
| | 비즈니스 인사이트 | 내 눈으로 해석한 비즈니스 현상 |
| | 기획 | 생각을 구조화하는 법 |
| | 경험 | 우리가 실제 세상에서 마주하는 서비스 |
| 감성 (4개) | 감정 | 내가 느끼는 것들 |
| | 크리에이티브 | 만들고 표현하는 것 |
| | 예술 | 아름다움을 발견하는 것 |
| | 취향 | 내가 좋아하는 것들 |
| 문화 (2개) | 컬쳐 | 사회와 문화의 흐름과 현상 |
| | 테크&트렌드 | 기술과 변화를 읽는 법 |
| 본질 (5개) | 철학 | 어떻게 살 것인가 |
| | 인간관계 | 사람 사이에서 나는 |
| | 루틴 | 무너지지 않는 일상을 설계하는 법 |
| | 라이프스타일 | 내가 추구하는 삶의 방식 |
| | 웰니스 | 건강한 나를 위한 몸과 마음 돌보기 |

## 4. 글감 설계 원칙

- 톤: 긍정·호기심·탐구 / 번아웃·탈진·무기력 전제 금지
- 구성: 감정·탐구형 70% + 구조화 유도형 30%
- 형태: 서술형 + 질문형 병행 / 구체적 장면 포착

## 5. Claude API 프롬프트 방향

### 구조 변경
| 구분 | 기존 (v2.0) | 변경 (v2.1) |
|---|---|---|
| 프롬프트 구조 | 단일 프롬프트, 카테고리만 변수 교체 | 카테고리별 16개 개별 프롬프트 |
| 톤·예시 | 공통 적용 | 카테고리마다 따로 설계 |
| 금지 조건 | 부정·결핍 중심 금지 | 번아웃·탈진·무기력 등 부정적 상태 전제 추가 금지 |

### 공통 프롬프트 원칙 (16개 공유)
- 독자: 20대 후반~30대 / 커리어 성장·창업 관심·내면 탐구
- 글감 정의: 읽는 순간 "나도 이런 적 있는데" 또는 "나는 어떻지?"가 즉각 튀어나오는 출발점
- 톤: 긍정·호기심·발견 / 구체적 장면
- 금지: 부정·결핍·번아웃·탈진·무기력 전제 / 뻔한 자기계발 / 추상적 감정 / 결론 내버리기
- 구성 비율: 감정·탐구형 70% + 구조화 유도형 30%

## 6. DB 스키마

| 테이블 | 주요 컬럼 | RLS |
|---|---|---|
| users | id, email, onboarding_completed | 본인만 |

|---|---|---|

| is_removed | boolean | 선택 후 취소 여부 |
| selected_at | timestamptz | 최종 선택 시각 |

## 8. 온보딩 UX (✅ 완료)

| 항목 | 확정 내용 |
|---|---|
| 브랜드 | spark (소문자) + "생각이 글이 되는 순간" |
| 히어로 | "어떤 주제로" (font-weight 500) / "글을 쓰고 싶나요?" (font-weight 400, 흐리게) |
| 카테고리 | 16개 4열 균등 그리드 |
| 카테고리 순서 | Fisher-Yates 셔플 (진입 시마다 무작위) |
| 카드 애니메이션 | translateY -3px / 2.4s / 선택 시 멈춤 + 블랙 반전 |
| 선택 조건 | 3개 이상, 상한 없음 |
| 버튼 | 원형 / 비활성: 회색 테두리만 / 활성: 블랙 채움 |
| 버튼 위 문구 | 0개: "3개 이상 선택해주세요" / 1~2개: "N개 더 선택해주세요" / 3개~: "✦ N개 선택됨" |
| Safe Area | 상단 59px / 하단 34px (iPhone 기준) |
| 스킵 | 없음 (강제 선택) |
| 최대 너비 | 390px 중앙 고정 (데스크탑 대응) |
